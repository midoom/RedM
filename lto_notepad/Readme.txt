Personal NotePad
Version: 1.0

English: This script allows you to write personal notes in a Notepad !
Français: Ce script vous permet d'Écrire des notes personnel dans un NotePad !

**Feature**:
- Inventory NotePad with Animation
- Save Notes in the Pad until the server reboot or the Player disconnect.
- Clear Button
- Auto Close the inventory when using

**How to Use**:
- add to your inventory the item: notepad 
- open inventory and use the item: notepad

**Change the background image**:
lto_notepad\html\index.css on the lines 19
